compute_3d_moment(n=0, l=0, m=0, DICT_ON=True)[source]

This function computes the Z_nl^m Zernike moment and stores it in a dictionary if DICT_ON = True (default).

Note 1) Saving the moments in the dictionary is high memory consuming.

Note 2) Each moment is one of the orthonormal basis vector of the Zernike expansion. Given a 3D function, f(x,y,z) defined in the unitary sphere, it can be decomposed in the Zernike basis as: f(x,y,z) = sum_nlm c_nlm Z_nlm

Note 3) Z_nl^m = (-1)^m conjugate(Z_nl^m)

compute_3d_coefficient(f, n, l, m)[source]

This function computes the Zernike coeffient associated to the Z_nlm moment as

c_nlm = int_(R<1) dxdydz F(x,y,z) * conjugate(Z_nlm)

Note that since we have voxelized the space, the integral becomes a sum over the voxels divided by the number of voxels (the voxels inside the R = 1 sphere).

bracket(z1, z2)[source]

This function computes the braket as c = < Z1 | Z2> = int dxdydz Z1 * conjugate(Z2)

zepyros.isolate_surfaces(surface, min_d=1.0)[source]

Given a surface of x, y, z points, form groups of points that are at a distance less than a given threshold

Parameters:

  • surface (ndarray) – array of points where each row is a point in space. If the number of columns is greater than three, only the first three columns will be used

  • min_d (float) – the minimum distance at which two points must be located to be grouped in the same cluster. The unit of measurement depends on that of the points in surface

Returns:

list of integer (starts from 2), where each integer represents the label of a specific group. The i-th element indicates the group to which the i-th point of the initial surface belongs

Return type:

array

Examples

>>> mat = np.array([[4.5, 4.5, 4.5], [1.5, 1.5, 1.5], [2., 1., 1.5], [5., 5., 5.], [1., 1., 1.]])
>>> mat
array([[4.5, 4.5, 4.5],
       [1.5, 1.5, 1.5],
       [2. , 1. , 1.5],
       [5. , 5. , 5. ],
       [1. , 1. , 1. ]])
>>> isolate_surfaces(mat, 1)
array([2, 3, 3, 2, 3], dtype=int8)
